# 🛡️ Is This Scam?
### India's AI-Powered Scam Detector

Built by **Sahina Khatun** & **Madhumita Saha** — two students from Kolkata 🇮🇳

---

## 🎯 What It Does
Paste any URL, UPI ID, phone number, job offer, or WhatsApp message — get a **Scam Probability Score (0-100%)** with explanation in **Bengali, Hindi & English** in seconds.

## ⚙️ Tech Stack
- **Frontend** → React.js + Tailwind
- **Backend** → Python FastAPI
- **AI** → Groq + Llama 3.1 (free)
- **Domain Check** → RDAP Protocol
- **Databases** → MHA Cybercrime, CERT-In, RBI, OpenPhish

## 🚀 Features
- 🔗 URL Phishing Detection
- 💳 UPI & Phone Fraud Check
- 🧠 AI Text Analysis
- 🌐 3-Language Output (English, Hindi, Bengali)
- 📸 Screenshot OCR (coming soon)

## 🛠️ Run Locally
```bash
# Frontend
npm install
npm start

# Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
